GINIJEV KOEFICIENT

Ginijev koeficient je merilo za statistično disperzijo, največkrat uporabljeno kot merilo neenakomerne razporeditve dohodka in premoženja. Definiran je kot razmerje z vrednostmi 0 in 1.

GINI = 0

Nižji ko je koeficient, bolj enakomerna je razporeditev.

Število 0 predstavlja popolno enakost (vsakdo ima popolnoma enak prihodek in premoženje).

GINI = 1

Višji ko je koeficient, bolj neenakomerna je razporeditev.

Število 1 predstavlja popolno neenakost.
